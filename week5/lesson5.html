    Notes

    1. Debugging easy definition : the process of identifying and removing errors from computer hardware or software.
      This debugging process often takes a long time, as fixing some errors may introduce others.

    2. Testing easy definition: Testing is the process to find bugs and errors.
      *There is no need of design knowledge in the testing process.
      *Testing can be manual or automated.
      *Testing is initiated after the code is written.

    3. Breakpoints: Breakpoints play an important role in debugging.
      They pause (break) a program when it reaches a certain point.
